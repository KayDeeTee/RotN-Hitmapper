# RotN-Hitmapper
hitmapper for rift of the necrodancer

## In track select
F1 => Auto play all

F2 => Auto play easy

F3 => Auto play medium

F4 => Auto play hard

F5 => Auto play impossible

# Installation
install BepInEx
put this in `BepInEx/plugins/hitmapper`
create a folder next to `RiftOfTheNecroDancer.exe` called `Charts`
